# PPT 风格提示词 - Blueshift 托管系统

## 🎨 整体风格设定

**主题定位：** 温馨可爱、轻松易懂的技术分享
**目标受众：** Solana 开发者、区块链学习者
**演讲时长：** 30-45 分钟

## 🌈 色彩方案

### 主色调（温馨柔和）
- **柔和粉色** `#FFB6C1` - 强调重点、按钮
- **薄荷绿** `#98FF98` - 成功状态、代码块背景
- **淡蓝色** `#B0E0E6` - 标题背景
- **奶油黄** `#FFFACD` - 警告、提示信息

### 辅助色
- **暖灰色** `#F5F5F5` - 背景色
- **深灰色** `#333333` - 主文字
- **中灰色** `#666666` - 次要文字
- **白色** `#FFFFFF` - 卡片背景

## 🎯 字体设计

### 中文字体
- **标题**：阿里巴巴普惠体 Bold / 思源黑体 Bold（圆润、可爱）
- **正文**：阿里巴巴普惠体 Regular / 思源黑体 Regular（清晰、易读）
- **代码**：JetBrains Mono / Fira Code（等宽、专业）

### 字号层级
- **主标题**：48-60pt
- **二级标题**：36-42pt
- **三级标题**：28-32pt
- **正文**：20-24pt
- **注释**：16-18pt

## 🖼️ 视觉元素

### 可爱图标风格
- **线条风格**：圆润线条（3-4px 圆角）
- **图标类型**：
  - 🔒 安全锁（金色，圆角矩形）
  - 💰 钱袋（橙粉色，卡通化）
  - 📦 礼物盒（彩色，带蝴蝶结）
  - 🤝 握手（暖色调，简化版）
  - ⚡ 闪电（黄色，活力感）

### 插画风格
- **扁平化 2.0**：带轻微阴影和渐变
- **柔和渐变**：从浅到深的同色系渐变
- **圆角设计**：所有矩形元素使用 12-16px 圆角

## 📐 布局规范

### 整体布局
- **边距**：左右 80px，上下 60px
- **卡片间距**：40px
- **内容区域**：居中对齐，最大宽度 1200px

### 卡片设计
```css
.card {
  background: #FFFFFF;
  border-radius: 16px;
  padding: 32px;
  box-shadow: 0 8px 24px rgba(0,0,0,0.08);
}
```

### 代码块样式
```css
.code-block {
  background: linear-gradient(135deg, #E8F5E9 0%, #F1F8E9 100%);
  border-radius: 12px;
  padding: 24px;
  font-family: 'JetBrains Mono';
  font-size: 18px;
  border-left: 4px solid #98FF98;
}
```

## 🎭 动画效果

### 页面切换动画
- **淡入淡出**：0.4s ease-in-out
- **滑动效果**：从右侧滑入，0.5s cubic-bezier

### 元素动画
- **图标**：轻微放大效果（scale 1.1）
- **列表**：逐项淡入（0.1s 间隔）
- **强调**：脉冲效果（关键信息）

## 📊 图表设计

### 流程图
- **节点样式**：圆角矩形，带可爱图标
- **连接线**：贝塞尔曲线，渐变色
- **箭头**：圆滑的三角箭头

### 状态机图
- **状态节点**：圆形或圆角矩形
- **转换箭头**：带动画效果的路径
- **颜色编码**：
  - 绿色 = 成功
  - 橙色 = 进行中
  - 红色 = 失败/取消

### 资金流动图
- **账户图标**：可爱的钱包或卡片
- **代币**：金色硬币图标
- **流动箭头**：虚线动画

## 🌟 特殊元素

### 高亮框
- **样式**：虚线边框，柔和的粉色背景
- **用途**：强调重要概念、核心代码

### 提示框
- **💡 小贴士**：黄色灯泡图标
- **⚠️ 注意**：橙色警告三角形
- **✅ 要点**：绿色对勾
- **❌ 避免**：红色叉号

### 页码指示
- **样式**：可爱的小圆点
- **当前位置**：实心，主题色
- **其他页面**：空心，灰色

## 🎬 演讲节奏

### 开场（2-3 分钟）
- 欢迎页 + 自我介绍
- 问题引入 + 解决方案概览

### 核心内容（25-35 分钟）
- 15 张幻灯片
- 每张 1.5-2.5 分钟
- 包含互动问答

### 总结（3-5 分钟）
- 要点回顾
- Q&A
- 资源链接

## 📱 响应式设计

### 投影模式（16:9）
- **推荐分辨率**：1920x1080
- **字号**：按照上述标准

### 便携模式（4:3）
- **调整字号**：整体缩小 10%
- **简化元素**：移除装饰性图标

## 📝 注意事项

1. **保持简洁**：每页不超过 5 个要点
2. **视觉层次**：使用大小、颜色、粗细区分重要性
3. **留白充足**：不要过度拥挤
4. **一致性**：保持风格、颜色、字体统一
5. **可读性**：确保后排观众能看清文字
